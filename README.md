# Tetris Server

A simple multiplayer Tetris server using Python and WebSockets.

## Files

* **`tetris.py`**: The game server and logic.
* **`constants.py`**: Game rules and shapes.
* **`index.html`**: The web client to play the game.

## How to Run

1.  **Install the requirement:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Start the server:**
    ```bash
    python tetris.py
    ```

3.  **Play:**
    Open http://localhost:8080 in your browser.
